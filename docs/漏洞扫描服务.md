# 漏洞扫描器

## 📖 概述

读取HTTP请求，导入开始漏洞扫描。采用**无害化检测策略**，不发送恶意payload，只做基础安全检测。

### 检测原则

- ✅ **完全无害化** - 不包含任何恶意攻击代码
- ✅ **不执行实际攻击** - 不触发WAF，不读取敏感信息
- ✅ **对目标无害** - 不对目标系统造成任何危害
- ✅ **需人工验证** - 所有检测结果需人工验证

### HTTP请求格式

```json
{
   "url": "http://127.0.0.1/test631?name=1",
   "method": "POST",
   "headers": {
       "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
   },
   "body": "aa=1&bb=a"
}
```

### 参数列表格式

```json
[
    {"param_name": "name", "param_value": "1", "param_type": "int"},
    {"param_name": "aa", "param_value": "1", "param_type": "int"},
    {"param_name": "bb", "param_value": "a", "param_type": "string"}
]
```

**参数类型**：
- `int` - 数字型（整数、浮点数）
- `string` - 字符型（普通字符串、JSON、哈希等）
- `url` - URL型（自动识别编码过的URL）

## 🚀 快速开始

### 基础使用

```python
from service.scaner import VulnerabilityScanner

# 创建扫描器实例
scanner = VulnerabilityScanner(dnslog_domain="your-dnslog.com")

# 构造HTTP请求
test_request = {
    'url': 'http://example.com/api?id=1',
    'method': 'GET',
    'headers': {'User-Agent': 'Mozilla/5.0'}
}

# 执行扫描
vulnerabilities = scanner.scan(test_request)

# 查看结果
for vuln in vulnerabilities:
    print(f"[{vuln['vuln_type']}] {vuln.get('param', 'N/A')}")
```

### 批量扫描

```python
request_list = [
    {'url': 'http://example.com/api1?id=1', 'method': 'GET', 'headers': {}},
    {'url': 'http://example.com/api2?name=test', 'method': 'GET', 'headers': {}},
]

results = scanner.scan_batch(request_list)
print(f"扫描了 {results['total']} 个请求，发现 {results['vuln_count']} 个漏洞")
```

### 自定义扫描选项

```python
scan_options = {
    'xss': True,
    'sqli': True,
    'rce': True,
    'ssrf': True,
}

vulnerabilities = scanner.scan(test_request, scan_options)
```

## 🎯 手动扫描模式详细说明

### 已修复的问题

#### Burp模式URL处理
**问题**: Burp格式中的URL通常是相对路径（如 `/api/test?a=1`），而不是完整URL。

**解决方案**:
- 自动检测URL是否为相对路径
- 从Host头部提取域名
- 组合成完整URL
- 自动识别HTTPS（端口443）

### 支持的格式

#### 1. json格式
```json
{
    "method": "POST",
    "url": "http://example.com/api/login?user=admin",
    "headers": {
        "User-Agent": "Mozilla/5.0",
        "Content-Type": "application/x-www-form-urlencoded"
    },
    "body": "username=test&password=123456"
}
```

#### 2. Burp模式（相对路径）
```http
POST /api/login?user=admin HTTP/1.1
Host: example.com
User-Agent: Mozilla/5.0
Content-Type: application/x-www-form-urlencoded
Cookie: session=abc123

username=test&password=123456
```

**自动转换为**: `http://example.com/api/login?user=admin`

#### 3. Burp模式（完整URL）
```http
GET http://example.com/api/test?a=1&b=2 HTTP/1.1
User-Agent: Mozilla/5.0

```

#### 4. Burp模式（HTTPS）
```http
POST /secure/api HTTP/1.1
Host: secure.example.com:443
User-Agent: Mozilla/5.0

data=test
```

**自动转换为**: `https://secure.example.com:443/secure/api`

**原因**: 缺少Host头部
**解决**: 在Burp请求中添加Host头部

## 🔬 漏洞检测策略（无害化）

### 1. XSS检测

**Payload**: `test631'">`

**检测内容**:
- 仅检测参数值是否被回显到响应中
- 不包含 `<script>`、`onerror` 等恶意XSS代码

**检测逻辑**:
```python
payload = "test631'\">"
# 发送请求
# 检测响应中是否包含 "test631'" 或 "test631'\">"
# 如果回显，则可能存在XSS，需人工验证
```

**风险等级**: 低风险 - 仅检测到参数回显，需人工验证

---

### 2. SQL注入检测

**Payload**:
- `'` (单引号)
- `''` (双引号)
- `'like'` (like语句)

**检测内容**:
- 仅检测是否触发SQL错误信息
- 不执行任何实际的SQL注入操作

**检测逻辑**:
```python
payloads = ["'", "''", "'like'"]
# 发送请求
# 检测响应中是否包含SQL错误信息
# 如: "SQL syntax", "mysql_fetch", "ORA-"等
```

**风险等级**: 低风险 - 仅检测到SQL错误，需人工验证

---

### 3. 命令执行检测（RCE）

**Payload**:
```
`xcmm=curl;$xcmm dnslog.com/l0g1r`
'`xcmm=curl;$xcmm dnslog.com/l0g1r`
"`xcmm=curl;$xcmm dnslog.com/l0g1r`
```

**检测内容**:
- 仅发送无害的DNS查询请求
- 不执行 `id`、`whoami` 等实际系统命令

**检测逻辑**:
```python
payload = "`xcmm=curl;$xcmm dnslog.com/l0g1r`"
# 发送请求
# 检查DNSLog平台是否收到DNS请求
# 如果收到请求，说明存在命令执行漏洞
```

**风险等级**: 无风险 - 仅发送DNS查询请求

---

### 4. SSRF检测

**Payload**: `http://{unique_id}.dnslog.com`

**检测内容**:
- 仅发送无害的DNS查询请求
- 不使用 `file://`、`dict://` 等可能读取敏感文件的协议

**检测逻辑**:
```python
payload = f"http://ssrf{timestamp}.dnslog.com"
# 发送请求
# 检查DNSLog平台是否收到DNS请求
# 如果收到请求，说明存在SSRF漏洞
```

**风险等级**: 无风险 - 仅发送DNS查询请求

```python
params = handler.callback_list_param(request_data)
```

```python
modified = handler.set_param_value(request_data, 'name', 'test631', mode=0)
```

```python
value = handler.get_param_value(request_data, 'name')
```

#### get_param_type - 获取参数类型

```python
param_type = handler.get_param_type(request_data, 'name')
```
