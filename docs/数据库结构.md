#### HTTP临时表
**表名**：`project_{name}_traffic`

**用途**：存储从mitmproxy捕获的原始HTTP请求，等待进一步处理

**字段**：
```json
{
  "url": "https://a.molun.com/auth/getAuthCodeInfoByCode",
  "headers": {
    "accept": "application/json, text/plain, */*",
    "content-type": "application/x-www-form-urlencoded",
    "user-agent": "Mozilla/5.0...",
    "cookie": "auth_sid=20000"
  },
  "method": "POST",
  "body": "code=1&client_id=65407",
  "time": "2025-11-14 17:20:26",
  "website": "http://a.molun.com/",

  "scaner_status":0,
  "source": 0  // 0=流量捕捉, 1=url生成
```

### 漏洞管理

#### 漏洞表
**表名**：`project_{name}_vuln`
**用途**：存储漏洞扫描发现的漏洞信息
**字段**：
```json
{
   "url": "http://127.0.0.1/test631?name=1",
   "method": "POST",
   "headers": {
       "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
   },
   "body": "aa=1&bb=a",
   "website":"http://127.0.0.1/",
   "subdomain":"127.0.0.1",
   "vuln_type":"xss",
   "vuln_type_detail": "xss-reflect",
   "level": "high",
   "paramname": "bb",
   "time": "2025-06-01 12:00:00"
}
```

#### 漏洞参数去重表
**表名**：`project_{name}_vuln_param`
**用途**：存储漏洞扫描发现的漏洞参数信息
**字段**：
```json
{
  "website": "http://127.0.0.1/",
  "url_path": "http://127.0.0.1/{string-7}",
  "param_name_list": ["name","aa"],
  "time": "2025-06-01 12:00:00"
}
```

#### 漏洞盲打log表
**表名**：`project_{name}_vuln_log`
**用途**：存储漏洞盲打测试的日志信息
**字段**：
```json
{
  "vuln_hash": "das3dadas",
```
