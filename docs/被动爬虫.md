# 被动爬虫

### 被动爬虫的原理

被动爬虫的原理是通过用户的操作来触发cdp，被动爬虫就会获取这个页面的信息，
chrome cdp调用
在webui处指定一个站点如https://www.molun.com/，然后cdp连接chrome，获取cdp tab，符合是站点的，就获取相关页面信息，如页面title、js获取url、页面内容等。

### 前端设计

1. 用户输入URL：提供一个输入框，用户可以输入要获取的网站URL。
2. 分为手动爬取与自动爬取，一个手动爬取按钮，一个自动爬取按钮。
3. 多个显示模块，cdp连接后，获取的所有tab符合url条件的信息。
4. 显示模块分为url与title信息、获取的页面a标签连接信息、从数据库中捕捉的流量请求信息、从数据库中获取的html信息。
4. 显示结果：展示爬取到的页面信息，如标题、URL、等。
