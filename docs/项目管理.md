# 项目管理

与后端 mongodb project_config 连接

## 功能列表

1. 项目管理，同时只能运行一个项目，status_code 0为未运行，1为运行中。
2. web前端要能看见项目运行状态，运行中，未运行，项目总数。
3. 项目新增、项目编辑、项目启动、项目关闭、项目删除功能。
4. 项目新增与编辑的表单，两种模式，正常表单编辑与编辑json。

## 项目目录结构

### 目录创建时机

在**项目启动时**自动检查并创建目录结构。

### 目录结构

```
project_data/
└── {项目名}/
    ├── images/              # 图片存储
    ├── logs/                # 日志文件
    ├── chrome_data/         # Chrome 数据目录
    │   ├── headless/        # 无头模式 Chrome 数据
    │   └── normal/          # 普通模式 Chrome 数据
    ├── whitelist_domain.txt # 域名白名单
    ├── blocklist_domain.txt # 域名黑名单
    └── blocklist_url.txt    # URL黑名单
```

### 文件说明

| 文件/目录 | 用途 |
|----------|------|
| `images/` | 存储爬虫抓取的图片等资源 |
| `logs/` | 存储项目运行日志 |
| `chrome_data/headless/` | 无头模式 Chrome 用户数据（CDP 19227） |
| `chrome_data/normal/` | 普通模式 Chrome 用户数据（CDP 19228） |
| `whitelist_domain.txt` | 域名白名单，每行一个域名 |
| `blocklist_domain.txt` | 域名黑名单，每行一个域名 |
| `blocklist_url.txt` | URL 黑名单，每行一个 URL |

---

## 项目删除

删除项目时会自动清理：
- 数据库中的项目配置
- `project_data/{项目名}/` 整个目录（包括 chrome_data）

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/projects/list` | GET | 获取项目列表 |
| `/api/projects/add` | POST | 添加项目 |
| `/api/projects/update` | POST | 更新项目 |
| `/api/projects/delete` | POST | 删除项目 |
| `/api/projects/start` | POST | 启动项目（自动创建目录） |
| `/api/projects/stop` | POST | 停止项目 |
| `/api/projects/status` | GET | 获取项目状态 |
| `/api/projects/count` | GET | 获取项目总数 |
